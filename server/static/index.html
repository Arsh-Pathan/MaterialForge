Inverse materials design in a compact, inspectable RL environment

MaterialForge turns materials discovery into an interactive lattice-construction problem. The agent must build a believable atomic arrangement that matches target macro-properties while respecting step and cost constraints. Featuring Percolation Threshold simulation and Gibbs-Stability heuristics.

Why this benchmark

Design intent

The environment is designed with high-fidelity scientific benchmarks in mind, including coordination number stability, positional entropy, and mirror symmetry. It features a non-linear percolation threshold for electrical conductivity—a first in the OpenEnv ecosystem.

8x8 state space Cost constrained Scenario driven

What judges can inspect

The playground exposes step-by-step lattice editing, reward movement, phase changes, and property traces. That makes the benchmark easier to evaluate than a hidden backend-only environment.

Reward

What the agent is optimizing

Reward combines property match, stability, lattice quality, and phase bonus, then subtracts cost pressure. This encourages good materials rather than just dense grids.

Reward Formula

reward = 0.50 * property_match
       + 0.25 * stability
       + 0.15 * lattice_quality
       + 0.10 * phase_bonus
       - cost_penalty

Quick Start

from material_forge_env import MaterialForgeAction, MaterialForgeEnv

with MaterialForgeEnv(base_url="https://arshpathan-material-forge-env.hf.space") as env:
    result = env.reset()
    action = MaterialForgeAction(action_type="place", row=0, col=0, atom="A")
    result = env.step(action)

Core Loop

1. Reset with a scenario and difficulty.

2. Place, replace, or remove atoms on the lattice.

3. Observe reward, phase, current properties, and score breakdown.

4. Stop when the target is hit or the budget is spent.

Difficulty ladder

Why three difficulty levels exist

Easy gives more budget headroom for exploration. Medium is the default interactive profile. Hard reduces both budget and tolerance so the agent has less room to waste actions.
